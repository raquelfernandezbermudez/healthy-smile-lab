# HealthySmile Lab 🎃
Aplicación interactiva de escritorio desarrollada con p5.js, MediaPipe FaceMesh y Electron.

## Descripción
La aplicación utiliza la cámara para detectar la apertura de la boca y simular un cepillado dental  mediante partículas, animaciones y recursos visuales con temática Halloween.

## Tecnologías
- p5.js
- MediaPipe FaceMesh
- Electron

## Estructura
- `index.html`
- `style.css`
- `sketch.js`
- `main.js`
- `assets/`
